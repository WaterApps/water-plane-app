About the app

The purpose of this app is to provide a visualization of DEM files loaded on your SD card. If there are none, a sample file will be loaded. A slider at the bottom lets you set the elevation, below which everything will be shaded in. This is handy for quickly finding the lowest and highest points in an area.

How to get DEMs

Markers

To add a marker, click the 'Add Marker' button in the top right corner then tap on the map. They can be dragged, removed, or have their text visibility toggled.

Location Tracking

When enabled, your location is set by GPS. When disabled, it is set by a draggable marker.

Follow Elevation

When this mode is enabled the water level will be set to match your current elevation.

Drawing Modes

The DEM can be displayed as all blue or with a color gradient, with optional transparency for both. The transparency level can be adjusted in the settings menu.
